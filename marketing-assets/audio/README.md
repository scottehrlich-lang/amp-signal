# AMP Signal — Pitch Narration Audio

Audio files referenced by `/pitch/index.html` via the `data-audio` attribute on each section. Generated by OpenClaw → ElevenLabs from `narration.json` in the same directory.

## Filename pattern
`{section-id}.mp3` — e.g. `01-open.mp3`, `04-renewal.mp3`

## Sections
- `01-open` · See the signal first
- `02-noise` · 14 tabs problem
- `03-platform` · One platform, every role
- `04-renewal` · Before the renewal (upsell)
- `05-call` · Before the call (brief)
- `06-cycle` · Before the cycle (political)
- `07-math` · 30% composite
- `08-pilot` · One region, one quarter
- `09-signoff` · See it first, win it first

## How playback works
- Audio is OFF by default
- User taps the speaker icon top-right to enable
- On section-visible, that section's MP3 plays
- Files lazy-load on first need (`preload=none`)
- Missing files fail silently — site still works without audio
